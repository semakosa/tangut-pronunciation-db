<a name="tones"></a>
# 声调系统

西夏本土韵书沿袭等韵学传统，区分平声（..𗗔.nye¹..）、上声（..𗨁.phu²..）两大调类。在现代转写系统中（包括本方案），一般采用数字标记法，分别称作第1调和第2调，在音节右上角标 _¹_ 和 _²_ 表示。

对于**归调存疑**或**音值不明**的单字，西夏学的习惯是在声调的位置上放一个体系外标记（meta-notation），本方案沿例作 _?_。一个典型的例子是 ..𘝦.dźə?.. “行”，出现在常用虚词 ..𘝦𘃡.dźə?vi¹..（作格标记）中。为了教学发音方便，本方案规定此类读音不确定的字**一律读作上声**。

<a name="tones-monosyllables"></a>
## 单字调值

<paracols>
无论研究者的母语是否区分声调，为避免在字词记忆和文献校勘中产生混淆，发音时仍应严格区分单字调。主流观点（[西田龙雄 1964](#ref-nishida1964)；[荒川慎太郎 1999](#ref-arakawa1999)）认为，西夏语的单字声调为“平声高、上声低”。[龚勋 (2025)](#ref-gong2025) 则主张，平声的单字调实际上并非高平调，而是一个高降调。因此，本体系确定单字调如下：

* 平声（..𗗔.nye¹..）：高降调 **HL** [˥˨]；
* 上声（..𗨁.phu²..）：[龚勋 (2025)](#ref-gong2025) 认为最可能的调型是中平调 **M**，但不能排除异说。实际发音中不要求具体的调型，只需音高避开高调域，且调势不下降即可。

>>>

* **平声**: 可径代以普通话的第四声（如“骂” _mà_）。需注意普通话第四声末尾常带有嘎裂音（creaky voice），在读西夏文时应尽量避免，以免与 [紧元音 _-h_](#tense-vowels) 的习惯读法混淆。
* **上声**: 在教学中，上声的发音不必强求一致，只要既不高也不降，选择自己最舒适的方式即可。以普通话为例，除了第四声，其他三个声调都可以作为替代：一声（如“妈” _mā_，建议音高略低，类似东北、湖南或马来西亚口音）、二声（如“麻” _má_）、三声（如“马” _mǎ_，需避嘎裂音）。笔者个人惯用中平调末尾略升 .[˧˧˦].，这种调型见于多种嘉绒语支语言。

</paracols>

## 多音节词的调值

西夏语多音节词的具体调值如今已无从确考。在实际发音中，如若将单字声调机械拼合，不仅大概率与古音不符，还会导致发音僵硬，破坏语言的自然韵律。本方案参考亲属语言的常见情况，提出一种教学上的约定读法，其规则如下：

* **音高峰值位置的确定**：在多音节韵律单元中，排除轻读的附着成分后，
  * **左起第一个平声字**被视为突显音节，其音高应为**全词最高**，前后各字的音高应更低；
  * 如果全词所有字**均为上声**，则将全词**最后一个上声字**视为突显音节，其音高也应在全词中相对最高。
* **词末突显音节保留单字调型**：如果左起第一个平声字为全词最后一个非轻读音节，或者全词所有非轻读音节均为上升字，那么这个词末字同时为突显音节者，应完整保留其单字调型特征（平声为高降调，上声为中平调）。
* **附着成分（enclitic）轻读**：某些虚词或成音节词缀视为 enclitic，语流中应作轻读处理，或降为低调，或贴合前字的音高曲线。动词人称标记一律轻读，如 ..𗧓.-ŋa̱²..、..𘂆.-tsə¹.. 等，其他虚词和词缀则不作严格规定。

以上约定保留了较大的发音自由度，例如强弱重音落在哪个音节上，非突显音节的音高如何降低、调域如何缩窄、调形如何压缩，允许存在个体差异。只要确保“首平最高”，多音节词的整体调型可以、也应当根据各自的母语习惯和个人偏好进行调整。例如，按照笔者的个人习惯，词末非轻读音节即使前面还有平声字，按上述规则不算作突显音节，仍倾向于保留单字调型，只是幅度有所缩减。如 ..𗯨𗖻.rur¹puh¹.. 等“平平”类二字组，笔者个人读作 **H.ML** ，但如果保留下字完全压低，读作 **HL.L**、或 **H.L**，也完全符合上述约定。

较长的复合名词不必一口气读完，而可根据语义切分为多个韵律单元，每段的调型独立确定。例如，“如来应正等觉”在西夏文中写作 ..𗌮𗆐𗖵𗟭𗣼𗳦𗫨.rmor¹lleh²bu¹vwo̱²tśha²qa̱¹rdwi̱wr²..，可以分三段读作 ..𗌮𗆐𗖵𗟭𗣼𗳦𗫨.rmor¹lleh² bu¹vwo̱² tśha²qa̱¹rdwi̱wr²..。

# 主元音与小舌化对立

[龚勋 (2020)](#ref-gong2020)认为西夏语存在六个基本元音音位：./a/., ./e/., ./i/., ./o/., ./u/., ./ə/.。每个元音音位都有**普通态**（对应三等韵）和**小舌化态**（对应一等韵，音位上可记作 ./ʶ/.）的对立。在本方案中，小舌化元音用下加短横线表示（如 _a̱_、_e̱_ 等， `U+0331`）。尽管二等韵字在音位上也被分析为小舌化元音，但它们与一等韵的主要区别在于存在一个[二等介音](#grade-ii-medial)（_-a-_），这一点将在后文详细讨论。此外，本方案还保留了一个不可拆分的复元音 [_uo_](#vowel-uo)。

西夏语的小舌化 ./ʶ/. 是一种元音的附加调音特征，在发元音时，同时发出轻微的 .[ʁ]. 音，即法语或标准德语的 r 音。小舌化遵循普遍音理，伴随舌根后缩（RTR）和舌身后移，从而影响元音的音质。方括号内的国际音标表示一种可能的音值。需要特别说明的是，这些标音记录的是在普通元音的基础上叠加小舌化后所产生的**综合听感**。例如，_e̱_ 之所以记作 .[ɛ̠ʶ].，谓先发 .[e]. 本音，继以小舌化或的RTR，听感像 .[ɛ̠].，而非先作 .[ɛ̠]. 音后施以小舌化。

为了直观掌握两类元音的音值区别，可以尝试以下的声韵配对练习：将软腭声母（如 _k-_）与三等元音配对，小舌音声母（如 _q-_）与一等元音配对。但需注意，在西夏语音系中，理论上的 _\*qi̱_ 型音节会增生出二等介音，实际演变为 _\*qai̱_ 形式，因此 _i_ 和 _i̱_ 不适用于此项练习。

<a name="vowel-a"></a>
## _a_ .[ɐ]. 与 _a̱_ .[ɑʶ].


<paracols>
* 三等 _a_ .[ɐ].: 任何较高或较前的a类元音。实际发音可以在 .[ɐ ~ ɛ ~ æ ~ a]. 范围内选择。
* 一等 _a̱_ .[ɑʶ].: 小舌化的低、央/后、展唇元音。

| 三等 _a_              | 一等 _a̱_                    |
| :--------------- | :----------------------- |
| ..𗧤.pha¹.. .[pʰɐ˥˨]. "砍" | ..𗂙.pha̱¹.. .[pʰɑʶ˥˨]. "另外" |
| ..𗐜.gha².. .[ɣɐ˧˧˦]. "盖" | ..𘕿.ġha̱².. .[ʁɑʶ˧˧˦]. （处所助词） |
| ..𗋕.tha¹.. .[tʰɐ˥˨]. "那" | ..𗢳.tha̱¹.. .[tʰɑʶ˥˨]. "佛" |

>>>

音值参考典例为阿拉伯语中字母 ك _k**ā**f_ 与 ق _q**ā**f_ 名称里元音 _ā_ 的对立。

母语区分两类a音者，通常可以直接将母语对立关系套用于西夏语 _a_ 与 _a̱_ 之间。如能区分“前a” ./a ~ æ/. 和“后a” ./ɑ/.，如苏州话“擦” _tsh**ae**h_ 与“尺” _tsh**ao**h_，或某些法语口音中 _p**a**tte_ 与 _p**â**te_，则可直接用于西夏语 _a_ 与 _a̱_。另一种常见的区分方式是舌位较高、接近 schwa 的 ./ɐ/. 和舌位较低、发音饱满的 ./ɑ ~ a/.，如印地语/乌尔都语中的 _a_ 与 _ā_。对于英语使用者而言，这两种区分方式都适用：
* 三等 _a_ .[ɐ].: 多数英语口音中的 _c**a**t_ .[æ ~ a]. 和 _b**u**t_ .[ɐ].。
* 一等 _a̱_ .[ɑʶ].: 通用美式英语 _h**o**t_ .[ɑ].，RP 或英格兰南部 _h**ar**d_ .[aː ~ ɑː].。

对于普通话母语者，或其他母语原生缺乏两类a音对立者，建议：
* 三等 _a_：读作英语 _b**u**t_ 或 _c**a**t_ 的元音。
* 一等 _a̱_：读作母语中的a音，如“坝” _b**à**_。

</paracols>

<a name="vowel-e"></a>
## _e_ .[e]. 与 _e̱_ .[ɛ̠ʶ].


<paracols>
* 三等 _e_ .[e].: 半高前不圆唇元音。可带有i介音 .[je].。
* 一等 _e̱_ .[ɛ̠ʶ].: 小舌化的半低、前/近前不圆唇元音。

| 三等 _e_              | 一等 _e̱_                    |
| :--------------- | :--------------------- |
| ..𗤋.me¹.. .[me˥˨]. "没有"  | ..𗑉.me̱¹.. .[mɛ̠ʶ˥˨]. "眼睛" |
| ..𘟀.le².. .[le˧˧˦]. "看见"  | ..𘜞.tse̱².. .[tsɛ̠ʶ˧˧˦]. "暂时" |
| ..𗊴.se¹.. .[se˥˨]. "血"  | ..𗑗.se̱¹.. .[sɛ̠ʶ˥˨]. "干净、纯洁" |
>>>

母语区分半高 ./e/. 和半低 ./ɛ/. 两类e音者，可将其自然用于西夏语 _e_ 与 _e̱_ 之间的对立：如法语 _b**é**b**é**_ 与 _b**ê**t**e**_，标准德语 _g**eh**_ 与 _B**e**tt_。一等元音 _e̱_ 可以加上小舌化和轻微央化来进一步强调。

若需刻意强调 _e_ 与 _e̱_ 之间的区别，可采用前加半元音的发音方式：
* 三等 _e_：可加上i介音 .[je].，类似普通话的“别” _b**ié**_，或俄语 _п**е**нь_。
* 一等 _e̱_：可加上软腭滑音 .[ɰɛ̠ʶ].，类似俄语 _м**э**р_ 刻意强调发音时的情况。

普通话母语者若无其他语言发音经验，建议：
* 三等 _e_：代以普通话 _ie_ 韵母，如“别” _b**ié**_。
* 一等 _e̱_：模仿华北/西北口音中 _ai_ 韵母近乎 .[ɛ]. 的发音，如 “白” _b**ái**_，但读作单元音 .[pɛ].。
</paracols>

<a name="vowel-i"></a>
## _i_ .[i]. 与 _i̱_ .[ɪʶ].

<paracols>
* 三等 _i_ .[i].: 高位前不圆唇元音。可带有细音作 .[ji].。
* 一等 _i̱_ .[ɪʶ].: 小舌化的次高、前/近前不圆唇元音。

| 三等 _i_              | 一等 _i̱_                    |
| :--------------- | :--------------------- |
| ..𘃡.vi¹.. .[ʋʲi˥˨]. "做" | ..𗨻.vwi̱².. .[wɪʶ˧˧˦]. "成为" |
| ..𗤢.bi².. .[bi˧˧˦]. "低" | ..𗾔.bi̱².. .[bɪʶ˧˧˦]. "太阳" |
| ..𗭪.si².. .[si˧˧˦]. （助词） | ..𗁸.si̱².. .[sɪʶ˧˧˦]. "紧" |

>>>

三等 _i_ 尤其是需刻意强调时，可带上i介音 .[ji].，或处理为成音节半元音 .[j̩.]。很多语言中的i音都易见此类倾向，如普通话“蜜” _m**ì**_，或法语 _v**ie**_，读西夏语时无需刻意规避。

一等 _i̱_ 本方案建议的教学用约定发音如下：在 i 的基础上，附加一定的软腭化，舌身后缩，可伴随软腭滑音形成 .[ɰɪʶ]. 的听感，参考俄语 т**ы** 刻意强调时双元音化的音值（而非波兰语单元音 t**y**），或土耳其语的 _k**ıy**ma_。

普通话母语者若无其他语言发音经验，建议：
* 三等 _i_：代以普通话 _i_ 韵母，如“笔” _b**ǐ**_。
* 一等 _i̱_：模仿华北/西北口音中 _ei_ 韵母近乎 .[ɪ]. 的发音，如 “北” _b**ěi**_，但读作单元音 .[pɪ].。
</paracols>

<paracols>

# 声母

# 全音节范围超音段特征

<a name="tense-vowels"></a>
## 紧元音：_-h_

<paracols>
韵类序列中的**第一小循环**（龚煌城认为是 R.61 至 R.76）普遍认为具有一个标志性音系特征，一般称为**紧元音**。龚煌城使用元音下加点标记（如 `ạ`），而荒川则在音节末尾加 `-q`。在本方案中，此类音节以音节末尾加 _-h_ 表示，若有韵尾则缀于韵尾之后：如 ..𗅲.teṃh²..。

主流观点认为，“紧”音节带有某种程度的促音或紧喉特征，类似于汉语入声在某些方言中的喉塞式体现，或者丹麦语的“stød”。在没有更有力反证的情况下，本方案姑从众说，则紧音节具有以下特点：
* 发声态为僵声（stiff）至嘎裂（creaky），在音节末尾近乎一个弱喉塞音 .[-ʔ].。
* 在词末时，元音往往偏短促，并突然中断。

| 紧元音               | 对照                |
| :--------------------- | :------------------ |
| ..𗏡.qu̱h¹.. .[qʊ̰ʶ˥˧ˀ]. "后来"        | ..𗌭.qu̱¹.. .[qʊʶ˥˨]. "…的话"            |
| ..𘎞.na̱h¹.. .[nɑ̰ʶ˥˧ˀ]. "鼻涕"  | ..𗓰.na̱¹.. .[nɑʶ˥˨]. "深"          |
| ..𗦁.ġhai̱h².. .[ʁˤɜ̯ˤɪ̰˧ˀ]. "障碍"    | ..𗼜.ġhai̱².. .[ʁˤɜ̯ˤɪ˧˧˦]. "利益"   |
| ..𗉝.śih².. .[ɕḭ˧ˀ]. "草"          | ..𗶹.śi².. .[ɕi˧˧˦]. "去"         |

>>>

* 第一和第二小循环之间的确切分界尚无定论。例如，荒川慎太郎将第一小循环的终点定在 R.79，因此在荒川构拟中 R.77–R.79 也是紧元音（_-yeq2_, _-eq'2_, _-yeq'2_）。韵书末尾最后几个韵类，笔者称之为“垃圾桶”者，可能也包含一些紧喉韵。如 R.104，龚煌城体系不同资料或作非紧喉的 _-ũ_，或作紧喉的 _-ụ̃_。荒川将 R.102 拟作 _-woq2_。
* 本方案采 _-h_ 记法，有意对“紧音节”的确切语音性质保持模糊，以便为异说留有余地。若通说成立，则 _-h_ 应理解为东亚地区音节末喉塞音或喉塞化的常见拼法，如漳泉片的白话字/台罗拼音，或吴语的吴协方案，如“白”字白话字作 _pe̍h_，上海话（吴协）作 _bah_。

</paracols>

<a name="retroflex-vowels"></a>
## 卷舌元音：_r-...-r_

<paracols>
韵类序列**第二小循环**（龚煌城认为是 R.77 至 R.98）辖字，普遍认为具有全音节的卷舌化特征，称为带有**卷舌元音**。龚煌城和荒川都在音节末尾用 `-r` 表示。本方案则采用括号式标记 _r-_..._-r_。注意以下拼写规则：
* 若声母为 [_r-_](#initials-laterals-rhotics)（该声母仅与卷舌韵搭配），则不在 _r-_ 前再写 _r-_，例如 ..𘜘.rir¹.. “得”，不写作 \*_rrir¹_。
* 后置 _-r_ 标记若遇韵尾，则缀于韵尾后，例如 ..𗄑.rŋo̱ṃr².. “皆”，不写作 \*_rŋo̱rṃ²_。

这种拼写规则主要是为了便于整体认读，音节前后各有一个 _r_ 字母，表示音节整体具有一致的 r 音色。最可能的语音实现是舌尖性的真卷舌元音，对于母语中没有此类元音（英语、普通话等的儿化不算）者，实在难以区分。为了便于区分，在实际发音中可以在元音或音节后再加上一个“儿”音来强化此特征。

| _-er_ | ..𗇋.rmer².. .[me˞˧˧˦]. (名词化后缀) | ..𗷝.me².. （否定前缀） |

## 鼻冠音：_n-, m-, ŋ-_ /N-/
在韵书中，有所谓“平行列” (parallel columns) 韵类。龚煌城将这些韵类拟为长元音（如 `aa`，也用下加线表示，如 `a̱`），而荒川则使用不对应具体音值的撇号（`'`）标记。根据 [龚勋 (2021)](#ref-gong2021)的鼻冠音说，这类韵的特点是音节首有一个音段性的 /N-/。本系统根据根据主声母的不同，采用不同的拼法来表示这个鼻冠音：

## 二等介音 (“ʕayn介音”)：_-a-_ ./-ʕ-/.

二等字音位上属于小舌化元音 ([Gong 2020](#ref-gong2020))，与一等字类似，但区别在于多出一个“二等介音”。龚煌城将此介音写作 `-i-`，荒川则写作 `-y-`。本方案的处理与前人不同，认为此介音类似 _a_ 音，可能是一个咽音 ./ʕ/.，相当于阿拉伯字母 ع _ʕayn_  <sub>汉语称“阿尼、二尼”等</sub>，故称“ʕayn介音”。拼写上记作 _-a-_，后面的主元音仍属小舌化元音，故带下加横线：_aa̱_, _ae̱_, _ai̱_, _ao̱_, _au̱_, _aə̱_。

| GX    | IPA    | 近似音     | 示例                     |
| :---- | :----- | :------------ | :-------------------------- |
| _aa̱_  | [a̯ˤa]  | _aa_, _ea_    | ..𗞽.baa̱¹.. .[ba̯ˤa˥˨]. "裂开"    |
| _ae̱_  | [a̯ˤɛ]  | _ae_, _ai_    | ..𗒘.ġhae̱¹.. .[ʁˤa̯ˤɛ˥˨]. "真"     |
| _ai̱_  | [ɜ̯ˤɪ]  | _ei_, _əi_    | ..𗪺.ġhai̱¹.. .[ʁˤɜ̯ˤɪ˥˨]. "力量"    |
| _ao̱_  | [a̯ˤɔ]  | _ao_, _eo_    | ..𗀋.phao̱².. .[pʰa̯ˤɔ˧˧˦]. "蛇"    |
| _au̱_  | [ɜ̯ˤʊ]  | _ou_, _eu_    | ..𘔯.ŋqhau̱¹.. .[ɴqˤʰɜ̯ˤʊ˥˨]. "迎接" |
| _aə̱_  | [ɜ̯ˤə]  | _aə_, _eə_    | ..𘇹.ṣaə̱¹.. .[ʂɜ̯ˤə˥˨]. "线"    |

在特定的音系环境中，这个二等介音 `-a-` 是自动增生的，而非用于区分韵类。主要有以下两种情况：

1.  **小舌音后接 _i̱_**：小舌音声母（V-K 牙音 _q-, qh-, ġ-_；VIII-H 喉音 _ġh-_）不能直接后接小舌化元音 _i̱_。必须强制插入一个 _-a-_，形成 _Qai̱_ 式结构，如 ..𗣓.ġai̱¹.. “难”（而非 *\*ġi̱¹*）。其他例子包括 ..𗎍.qhai̱².. “讨厌”、..𗵒.qai̱¹.. “金子”、..𗙏.ġhai̱h².. “音”。
2.  **卷舌音声母之后**：卷舌音声母（IV-Ṇ 舌上音：_ṇ-_；VII-Š 正齿音：_tṣ-, tṣh-, dẓ-, ṣ-_；IX-L 来日音：_ẓ-_）不能直接后接任何一种小舌化元音，而必须插入二等介音 _-a-_。例如：..𘇹.ṣaə̱¹.. “线”，..𘓟.dẓae̱².. “相信”，..𗉋.tṣao̱ṃ¹.. “聚集”，..𘝣.ṇae̱ṃ¹.. “浑浊”。

<a name="yod-medial"></a>
## 细音介音（罕见）：_-y-_ ./-j-/.

西夏语三等字并不存在细音介音，只有 [_e_](#vowel-e) 和 [_i_](#vowel-i) 可以读作 _ye_ 和 _yi_，但这并非其区别于一等音节的标志。正因如此，为了满足转写其他语言的需求，西夏人特地造了一些包含 _-y-_ 介音的字。这些字通常是所谓切音字，由表示声母的字与 ..𘁂.ya².. 字组合而成，如：

* ..𘗔!dya²!.. （由 ..𘗐.de².. 和 ..𘁂.ya².. 组合而成）
* ..𗓮!mya²!.. （由 ..𗓁.me².. 和 ..𘁂.ya².. 组合而成）

这些字很可能仅见于佛教从业者使用的特定语体，不仅在音系中处于边缘地位，而且在语言共同体中也属于外围成分。
